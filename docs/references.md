# External References

- Main site: https://www.usaspending.gov
- API root + documentation index: https://api.usaspending.gov
- API endpoints reference: https://api.usaspending.gov/docs/endpoints
- API intro tutorial: https://api.usaspending.gov/docs/intro-tutorial

- Opportunity Management API (federal-account auth required): https://open.gsa.gov/api/opportunities-api/
- Assistance Listings API: https://open.gsa.gov/api/assistance-listings-api/

- SAM.gov Data Services (bulk + reps/certs mapping): https://sam.gov/data-services
- API key & account: https://sam.gov/workspace/profile/account-details
